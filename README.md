# 🌱 Smart Soil Moisture Irrigation System

## 📌 Project Overview
The **Smart Soil Moisture Irrigation System** is an IoT-based automation project designed to reduce manual effort in irrigation and save water.

This system uses an **Arduino Uno** and a **soil moisture sensor** to continuously monitor soil condition.

- When the soil is **dry**, the water pump automatically turns **ON**
- When the soil is **wet**, the pump automatically turns **OFF**

The system also sends real-time updates such as soil moisture value, soil condition, and pump status to the user through a **Telegram Bot**.

This project is highly useful for **small-scale farming and smart agriculture applications**.

---

## 🎯 Features
- Automatic irrigation control
- Real-time soil moisture monitoring
- Water pump automation
- Telegram notifications
- LED and buzzer indication
- Low-cost and simple implementation

---

## 🛠 Hardware Components
- Arduino Uno
- Soil Moisture Sensor
- 5V Single Channel Relay Module
- 5V Mini Water Pump
- Red LED
- Green LED
- Jumper Wires
- Breadboard
- 220Ω Resistors (2)

---

## 💻 Software Requirements
- Arduino IDE
- Telegram App
- Telegram Bot
- Circuito.io

---

## ⚙️ Working Principle
1. Soil moisture sensor reads soil condition
2. Arduino processes the sensor value
3. If soil is dry → pump turns ON
4. If soil is wet → pump turns OFF
5. Telegram bot sends real-time updates
